Google Drive Setup
One-time setup (~5 min). Uses Google Apps Script — no OAuth, no Client ID, just your Google account.
1Go to script.google.com → click New project
2Delete any existing code. Paste the contents of Code.gs into the editor → Save (Ctrl+S)
3Click Deploy → New deployment → type: Web app → Execute as: Me → Who has access: Anyone → Deploy
4Copy the Web app URL (starts with https://script.google.com/macros/s/...) and paste below
5Click Save & Test. The app will create Claude/Credit Card App/ in your Drive automatically on first save.
After deploying, if Google asks to review permissions, click Advanced → Go to (app name) → Allow.
